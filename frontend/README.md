# 💻 MedWise Frontend

## 📌 Overview
The **MedWise Frontend** is the user interface of the MedWise application. It allows users to easily search for medicines, view their generic alternatives, compare prices, and understand potential savings.

The frontend is designed to be **simple, responsive, and user-friendly**, ensuring that users can quickly access important medicine information and make informed decisions about affordable healthcare options.

The interface communicates with the **MedWise backend API** to fetch medicine data and display it in an organized and visually appealing format.

---

## 🎯 Purpose

The main goal of the MedWise frontend is to provide:

- 🔎 Easy **medicine search functionality**
- 💊 Clear **generic medicine information**
- 💰 **Price comparison** between branded medicines and Jan Aushadhi medicines
- 📉 Display of **maximum possible savings**
- 📋 Detailed **medicine information**

---

## ✨ Features

### 🔎 Medicine Search
Users can search for medicines using the search bar.

### 💊 Medicine Details Page
Displays detailed information about the selected medicine including:
- Generic name
- Category
- Description
- Strength and form

### 💰 Price Comparison
Shows:
- Brand medicine prices
- Jan Aushadhi price
- Maximum savings percentage

### 📋 Additional Information
Displays medical details such as:
- Indications
- Typical adult dosage
- Side effects
- Precautions

### 📱 Responsive UI
The interface is responsive and works smoothly on:
- Desktop
- Tablets
- Mobile devices

---

## 🛠️ Technologies Used

- **React.js** – For building the user interface
- **Tailwind CSS** – For styling and responsive design
- **Axios / Fetch API** – For making API requests
- **JavaScript (ES6+)** – Core frontend logic
- **HTML5** – Page structure
