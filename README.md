# 💊 MedWise – Smart Generic Medicine Finder

## 📌 Project Description
**MedWise** is a web-based healthcare application designed to help users **find affordable generic alternatives to branded medicines** and compare their prices across different sources. The platform focuses on improving **medicine affordability and accessibility** by promoting awareness about **generic medicines**, especially those available through **Jan Aushadhi stores**.

Many patients unknowingly purchase expensive branded medicines even though **cheaper generic versions with the same active ingredients** are available. **MedWise** solves this problem by allowing users to search for a medicine and instantly view:

- The **generic medicine name**
- **Brand alternatives**
- **Price comparison**
- **Potential savings**
- **Medical details such as dosage, side effects, and precautions**

The application provides a **simple and user-friendly interface** where users can quickly search medicines and make informed healthcare decisions.

---

## 🎯 Objectives

The main objectives of MedWise are:

- 💰 Help users **save money on medicines**
- 💊 Promote **generic medicine awareness**
- 🔍 Provide **easy medicine search**
- 📊 Show **price comparisons between branded and generic medicines**
- 🏥 Encourage the use of **Jan Aushadhi medicines**

---

## ⚙️ Key Features

- 🔎 **Medicine Search**  
  Users can search for medicines by name.

- 💊 **Generic Alternative Suggestions**  
  Displays equivalent generic medicines.

- 💰 **Price Comparison**  
  Shows prices from different brands and Jan Aushadhi stores.

- 📉 **Savings Calculation**  
  Displays the percentage of money users can save.

- 📋 **Detailed Medicine Information**
  - Indications
  - Dosage
  - Side effects
  - Precautions
  - Strength and form

- 📱 **Responsive UI**  
  Built with modern UI design to work across different devices.

## 🛠️ Technology Stack

### Frontend
- React.js  
- Tailwind CSS  
- Axios  

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB (Medicine database)

---

## 📂 Project Workflow

1. User searches for a medicine.
2. The frontend sends a request to the backend API.
3. The backend fetches medicine data from the database.
4. The application displays:
   - Medicine details
   - Generic alternatives
   - Brand prices
   - Savings information

---

## 🌍 Real-World Impact

MedWise aims to support **affordable healthcare** by helping people discover **low-cost generic medicines**. This is especially useful for:

- Patients with long-term medication needs
- Low-income communities
- Students and families looking to reduce medical expenses

By increasing awareness of **generic medicines and Jan Aushadhi programs**, MedWise contributes to **better healthcare accessibility**.
